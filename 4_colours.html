Exercise number four:


Context excersice:

Create a web page named 4_colors.html with the following characteristics.

  1. Create 4 divs with a height and width of 200px and the colors blue, red, green, and yellow (insert styles using the style attribute). Each div must have a unique identifier.
  2. Within the script, add an event so that when each div is clicked, its color changes to black. Use addEventListener to add the event.
  3. Create a page with a script that stores a color in a global variable depending on the keyboard letter pressed.
    • The letter 'a' will store the color pink.
    • The letter 's' will store the color orange.
    • The letter 'd' will store the color light blue.
    • To store the color, see the tip at the end of the instructions.
    • Create a new div with the id "key", 200px wide and 200px high, and with a white background and black border.
    • When the keys 'a', 's', or 'd' are pressed, the color of the div "key" should change to pink, orange, or light blue, respectively.
  4. Following the logic of the previous point, pressing the q, w, or e keys will create a new div of the same dimensions mentioned above with the colors purple, gray, and brown respectively.
